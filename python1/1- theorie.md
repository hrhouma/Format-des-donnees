### 1. **Impression et commentaires**
- **Imprimer quelque chose à l'écran :**
  ```python
  print("Bonjour le monde!")
  ```
- **Écrire un commentaire :**
  ```python
  # C'est un commentaire
  ```

### 2. **Variables et Types**
- **Créer une variable :**
  ```python
  ma_variable = 10
  ```
- **Variables de différents types :**
  ```python
  entier = 5
  flottant = 5.0
  chaine = "Hello"
  ```

### 3. **Opérations mathématiques**
- **Addition, Soustraction, Multiplication, Division :**
  ```python
  a = 5 + 3
  b = 5 - 3
  c = 5 * 3
  d = 5 / 3
  ```

### 4. **Listes**
- **Créer une liste :**
  ```python
  ma_liste = [1, 2, 3, 4, 5]
  ```
- **Accéder à un élément de la liste :**
  ```python
  premier_element = ma_liste[0]
  ```

### 5. **Instructions conditionnelles**
- **Utilisation de if, elif, else :**
  ```python
  if ma_variable > 10:
      print("Plus grand que 10")
  elif ma_variable == 10:
      print("Égal à 10")
  else:
      print("Moins que 10")
  ```

### 6. **Boucles**
- **Boucle for :**
  ```python
  for i in ma_liste:
      print(i)
  ```
- **Boucle while :**
  ```python
  i = 0
  while i < 5:
      print(i)
      i += 1
  ```

### 7. **Fonctions**
- **Définir une fonction :**
  ```python
  def ma_fonction():
      print("Hello, fonction!")
  ```
- **Appeler une fonction :**
  ```python
  ma_fonction()
  ```
